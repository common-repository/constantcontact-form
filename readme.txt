== Description ==

> **IMPORTANT NOTICE**: We are currently not maintaining  this plugin. We ***RECOMMEND you download the [Contact Form by ContactUs.com](https://wordpress.org/plugins/contactuscom/) plugin for WP***. So, instead of downloading this one, try [Contact Form by ContactUs.com](https://wordpress.org/plugins/contactuscom/) , or try adding the JavaScript snippet directly into your footer.php of your template, [Click here to learn how.](http://help.contactus.com/hc/en-us/articles/200676216-Getting-the-Website-Code)
